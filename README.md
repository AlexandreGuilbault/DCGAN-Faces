# DCGAN-Faces
Pytorch implementation of a DCGAN to generate faces from CelebA dataset

## Datasets

CelebA dataset from : https://www.kaggle.com/jessicali9530/celeba-dataset#img_align_celeba.zip is being used.

## Dependencies

Pytorch     0.4.1\
torchvision 0.2.1\
matplotlib  2.2.2\
numpy       1.15.1
